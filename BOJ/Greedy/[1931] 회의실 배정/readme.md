# [1931] 회의실 배정

## 분류💁

그리디

## 접근법
> 한 개의 회의실에서 최대로 할 수 있는 회의의 개수를 구하자

## 시퀀스
- 회의가 끝나는 시간을 기준으로 오름차순 정렬을 한다.
- 끝나는 시간이 가장 작은 것을 택하는 것이 회의가 최대가 될 수 있다.
- 회의 전체를 반복문으로 돌면서 `cur`(현재 끝나는 시간)보다 크면 `cur`값을 업데이트 해주고 회의개수를 +1해준다.
